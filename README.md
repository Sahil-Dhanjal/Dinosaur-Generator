This is my very first Nodejs Application that works on randomly generating dinosaur names, & their corresponding Images. I've made use of two APIs to receive responses to the request sent by the user via clicking a button. The APIs used were: <br/>

* `Dino Ipsum API` - *used for generating random dinosaur Names*. [Link](https://dinoipsum.com/).
* `Bing Image Search API` - *used for generating dinosaur Images*. [Link](https://rapidapi.com/microsoft-azure-org-microsoft-cognitive-services/api/bing-image-search1/). <br/>
